CampusConnect — Prototype de stage Tail’ed

---

Développé par : Saad NADI
Date : Mars 2025

Objectif :
CampusConnect est un prototype d’application web permettant aux étudiants :
- de consulter des annonces utiles sur leur campus
- de publier leurs propres annonces (événements, dons, entraide, etc.)

Le projet a été réalisé dans le cadre de ma candidature au stage chez Tail’ed.

---

Technologies utilisées :
- Frontend : React (avec Vite)
- Backend : Express.js (Node.js)
- UI : TailwindCSS
- API : REST (GET & POST)
- Base de données : simulée (en mémoire, via tableau JavaScript)

---
Fonctionnalités

- Liste dynamique des annonces
- Formulaire de publication
- Ajout d’annonces persistantes (sauvegardées en JSON)
- Rafraîchissement instantané après publication
